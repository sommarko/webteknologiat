Grid on 2 dimension tapa järjestää asioita verkkosivulle.
Esimerkiksi luoda yhdelle riville max 3 gridiä
css grid-template-columns: 2fr 2fr 1fr
Grid myös osaa tiivistää tekstin kuinka isoon tilaan vaan. Kokeile muuttamalla kokoa.
Kätevä työkalu luoda syvyyttä omiin nettisivuihin.
style kohdasta muuttamallla grid-template-columns kohtaa saadaa luootua eri tyyliä.
Esimerkiksi grid-template-columns: 1fr 2fr 1fr
saadaan luokitettua 3 gridin koot ja missä järjestyksessä.
myös grid-row komento antaa muutta paikkaa.
rowia muuttamalla saadaaan vaikutettuun gridien sisältöjen kohtaa sivulla.
